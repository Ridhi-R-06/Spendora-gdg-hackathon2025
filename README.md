# Spendora-gdg-hackathon2025
Spendora revolutionizes finance with AI "Weekly Micro-Cycles" on a Google stack (IDX, Firebase). Features include Gemini automation, TensorFlow.js forecasting, and real-time Group Connect. It motivates via gamification and ensures reliability with a "Safe Mode" fallback to the 50/30/20 rule, prioritizing emergency preparedness from day one.
